# Ralph Wiggum Task: Suno AI to MIDI FL Studio Automation

## Objective
Build a system that converts Suno AI audio to MIDI files and integrates with FL Studio for automated music production using open source tools.

## Requirements

1. **Suno AI to MIDI Conversion**
   - Research and implement open source tools for audio-to-MIDI conversion
   - Handle Suno AI output formats
   - Ensure quality MIDI extraction

2. **FL Studio Automation**
   - Create MCP tool for FL Studio desktop automation
   - Use Playwright or similar for UI automation
   - Enable track creation and mastering workflows

3. **Open Source Agent Framework**
   - Use Ollama with open source models for agent guidance
   - Consider LangChain agents with tools OR
   - Consider Claude Code SDK adapted for open source models
   - Prioritize open source solutions

4. **User Controls**
   - Implement start/stop functionality
   - Accept user prompts to control workflow
   - Provide status feedback

5. **Performance Optimization**
   - Use Claude subagents for parallel task execution
   - Use Claude skills where applicable
   - Use async subagents when possible

## Success Criteria
When all components are implemented and working together, output:
<promise>DONE</promise>

## Implementation Notes
- Prioritize open source tools throughout
- Focus on Windows desktop compatibility
- Create modular, maintainable code
- Document setup and usage clearly
